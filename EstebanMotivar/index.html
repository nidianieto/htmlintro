¡Bienvenido a la Práctica de Git!

¿Qué es Git?

Git es un sistema de control de versiones distribuido que te permite realizar un seguimiento de los cambios en tus archivos a lo largo del tiempo. Es ampliamente utilizado en el desarrollo de software para colaborar en proyectos y mantener un historial de modificaciones.

Beneficios de Git

  • Historial completo de cambios en tus archivos.
  • Facilita la colaboración en equipos de desarrollo.
  • Retroceso sencillo a versiones anteriores.
  • Creación de ramas para desarrollar nuevas funcionalidades sin afectar la versión principal.

Principales comandos de Git

Aquí hay algunos comandos esenciales de Git:

                
// Clonar un repositorio remoto a tu máquina local
git clone 

// Añadir archivos al área de preparación
git add 

// Confirmar cambios en el repositorio
git commit -m "Mensaje descriptivo"

// Subir cambios al repositorio remoto
git push origin 

// Actualizar tu repositorio local con los cambios remotos
git pull origin 

// Crear una nueva rama
git branch 

// Cambiar a una rama existente
git checkout
